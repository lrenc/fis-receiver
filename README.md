# fis-receiver

fis receiver with golang

## 安装
```
bash -c "$(curl -k https://raw.githubusercontent.com/lrenc/fis-receiver/master/install.sh)"
```

## 启动

```
cd fis-receiver

nohup ./main & # 默认端口8527 可自定义
```

## Test

浏览器访问 http://host:port

## 其他

机器无法访问外网？可以先将代码安装到可以访问的机器，然后执行scp，或者直接联系我。
